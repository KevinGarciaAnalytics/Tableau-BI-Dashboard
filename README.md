# Tableau-BI-Dashboard
Tableau Dashboard created to find specific information about the companies profit. 

## Project Overview
This project presents an interactive Tableau dashboard developed to analyze business performance through dynamic visualizations and key performance indicators (KPIs). The dashboard enables users to explore trends, compare performance across categories, and make data-driven business decisions.

## Project Objective
Create an interactive dashboard that transforms raw business data into meaningful visualizations, allowing users to monitor performance, identify trends, and analyze key business metrics.

## Tools & Technologies
- Tableau
- Microsoft Excel
- Data Visualization
- Dashboard Design
- Business Analytics

## Dashboard Features
- Interactive dashboard with filters and slicers
- KPI tracking and performance monitoring
- Trend analysis over time
- Category and regional performance comparisons
- Dynamic charts and visualizations
- User-friendly dashboard layout
